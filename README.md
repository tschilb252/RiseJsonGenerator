# RiseJsonGenerator
Codebase for generating RISE-formatted JSON files. This codebase can generate RISE files for any data that [Pisces](https://github.com/usbr/Pisces) can query from the list shown below:

- CPN Hydromet
- MB-ART Hydromet
- LCB HDB
- UCB HDB
- YAO HDB
- ECAO HDB
- KBAO HDB
- LBAO HDB
- USGS
- CA CDEC
- NRCS SNOTEL
- USACE NWD
- OWRD
- IDWR
- NOAA
